Lesson 1: The Beautiful Scenery

Focus & Try

Focus ①

「〜です」「〜がいます、あります」　be動詞の現在形

This is an aurora.

これはオーロラです。

be動詞は前後の語句をイコール（＝）で結んだり、主語の情報や存在を表したりします。
疑問文 Are you a baseball fan?

あなたは野球のファンですか。

否定文 I am not a baseball fan.

私は野球のファンではありません。

Focus ②

「〜します」　一般動詞の現在形

Mt. Fuji looks beautiful.

富士山は美しく見えます。

一般動詞は主語の動作や状態を表します。主語が三人称・単数のときは動詞のうしろに -(e)s をつけます。
主語 be動詞 一般動詞の例
単数
I am start / go / study
You are start / go / study
He / She / It / Mike など is starts / goes / studies
複数 We / You / They / Students など are start / go / study

Try 1

日本語に合うように，（　）に am, are, is のいずれかを入れましょう。

① There ( are ) some places with beautiful night scenery.

美しい夜景を見られる場所がいくつかあります。

② Hakodate ( is ) one of them.

函館はその１つです。

③ I ( am ) always surprised at the beauty.

私はいつもその美しさに驚いています。

Try 2

日本語に合うように，（　）内の語句を並べかえましょう。

① Many foreign ( visit / people / Mt. Fuji ).

→ Many foreign people visit Mt. Fuji.

たくさんの外国人が富士山を訪れます。

② They ( a sea of / see / clouds ).

→ They see a sea of clouds.

彼らは雲海を見ます。

③ That ( is / with / a place ) beautiful scenery.

→ That is a place with beautiful scenery.

そこは美しい景色が見られる場所です。

Try 3

日本語に合うように，最も適する語を下から選び，（　）に入れましょう。

artificial/ruins/romantic

① We see the castle ( ruins ).

私たちは城跡を見ます。

② The castle is ( artificial ), but very beautiful.

そのお城は人工的なものですが、とても美しいです。

③ At night, the castle with lights is ( romantic ).

夜にライトアップされたお城はロマンチックです。
